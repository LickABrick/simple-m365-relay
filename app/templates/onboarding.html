Welcome to Simple M365 Relay

A minimal Postfix relay for Microsoft 365 (OAuth2 / XOAUTH2), with a clean web UI.
What problem it solves
Send mail via Microsoft 365 without storing an SMTP password — using OAuth device flow and refresh tokens.
What you’ll configure
  • Relayhost + TLS policy
  • OAuth tenant/client IDs
  • Optional: device flow login
  • Optional: test mail + add SMTP AUTH users
Tip: you can skip the optional steps and finish quickly — you can always come back later.
